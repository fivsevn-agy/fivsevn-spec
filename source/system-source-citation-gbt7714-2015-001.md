# GB/T 7714-2015 Citation Format (Informative)
# GB/T 7714—2015 引用格式参考

- 与开放标准（如 RFC、APA Style）不同，GB/T国家标准沿用出版物体系（也就是说它属于书/期刊，因此有版权、有发行渠道、有售价，政府只负责“发布和备案”，不负责“免费普及阅读”），其全文获取受版权与发行机制限制，不以公开在线文档为默认形态（你可能无法找到官方的公开的详细说明文档）。
